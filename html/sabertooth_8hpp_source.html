15  * of H-Bridge motor controllers. These can be used to control permanent magnet DC motors
16  * between 6 & 30 volts, up to 25 amps continuous and 50 amps surge.
17  *
18  * This driver works in the "Packetized Serial" mode, which must be set with the DIP
19  * switches on the controller hardware. The driver defaults to operating only one
20  * Sabertooth, using the lowest physical address of 128. The address is configurable in
21  * the driver by assigning the address value to the constant "MOTOR_DRIVER_ADDRESS_x".
22  *
23  * The driver is written in the C language for Microchip PIC microcontrollers using CCS's
24  * PIC-C Compiler, and was developed using version 4.120 of that compiler. I have also used it
25  * with 32-bit PICs, using Microchip's C32 compiler.
26  *
27  * It should be easily convertible to other microcontrollers and compilers with minimal effort. I have
28  * annotated the three lines which would have to be modified for a different compiler or
29  * processor architecture.
30  *
31  * Dimension Engineering's web site is at: www.dimensionengineering.com
